# CadOwl Agent System

## Governance
- Governor Agent: enforces roadmap, safety, and release quality.
- Principal Engineer Agent: architecture integrity, modularity, debt control.
- Product Architect Agent: PRD, acceptance criteria, workflow coherence.

## Core Delivery Agents
- Backend API Agent
- Import/Export Agent
- Validation/Self-Healing Agent
- GIS/Coordinate Agent
- Cable/Topology Agent
- AI/MAXILLM Agent
- DORIS Intelligence Agent
- QA/Test Agent
- Documentation Agent
- Repo Automation Agent

## Required Agent Output Template
Every agent update must include:
1. What changed
2. Why changed
3. What remains
4. Risks/blockers
5. Next recommended action

## Safety Contracts
- No agent may claim success on failed validation/test
- Any destructive proposal requires rollback path
- AI recommendations must include confidence + evidence source
