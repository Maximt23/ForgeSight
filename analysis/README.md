# CadOwl Analysis Output Directory

This directory contains analysis outputs from CadOwl pattern detection improvement tasks.

## Expected Outputs

| File | Task | Description |
|------|------|-------------|
| `unmatched_patterns.json` | CADOWL-001 | Blocks/layers not matched by current detector |
| `accuracy_report.md` | CADOWL-002 | V1 vs V2 coordinate comparison analysis |
| `pattern_recommendations.md` | CADOWL-003 | Recommended pattern additions |
